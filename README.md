🚦 Road Accident Analysis Dashboard (India) | Power BI
📌 Project Overview

This project presents an interactive Power BI Dashboard that analyzes road accident data across 28 states of India. The dashboard provides insights into accident trends, severity, weather conditions, road types, vehicle involvement, and other important factors that contribute to road accidents.

The objective of this project is to transform raw accident data into meaningful visualizations that help identify patterns and support data-driven decision-making.
